# ITSound — Sistema Demo Submission

## Email Template per Risposta Automatica

Crea un'autoresponder col tuo gestore email (o plugin WordPress) per quando arriva una demo.

### Oggetto: Grazie per averci inviato la tua demo! 🎵

```
Ciao [Nome],

Grazie per aver inviato la tua demo a ITSound! 🎵

Abbiamo ricevuto il tuo materiale e il nostro team lo ascolterà con attenzione.

⏱ Ti risponderemo entro 7-10 giorni.

Cosa valutiamo:
- Qualità della produzione
- Originalità del sound
- Coerenza con la nostra identità

Nel frattempo, seguici su Instagram @it.sound_ per rimanere aggiornato sulle novità.

A presto,
Marco & Andrea
ITSound — Martina Franca
itsound.it
```

---

## Criteri per Valutare una Demo
*(da usare internamente)*

| Criterio | Peso | Valuta 1-5 |
|----------|------|------------|
| Qualità audio (produzione) | 30% | |
| Originalità / Personalità | 25% | |
| Coerenza col genere ITSound | 20% | |
| Potenziale commerciale | 15% | |
| Professionalità (presentazione) | 10% | |
| **TOTALE** | **100%** | |

**Esito:**
- **20+** → Contattare subito
- **15-19** → Tenere in considerazione, richiedere altro materiale
- **<15** → Gentile rifiuto
